# Educação Financeira com NotebookLM

Projeto desenvolvido como parte do desafio da DIO utilizando o NotebookLM como ferramenta de aprendizagem ativa para estudar conceitos fundamentais de educação financeira.

## Contexto

A educação financeira é uma habilidade essencial para a tomada de decisões conscientes sobre dinheiro, planejamento pessoal e construção de patrimônio. Neste projeto, utilizei o NotebookLM para organizar informações provenientes de fontes oficiais e explorar o tema de forma estruturada por meio de perguntas estratégicas, análise comparativa e construção de um miniguia de estudos.

## Objetivos de Estudo

* Compreender os conceitos fundamentais de educação financeira.
* Entender a importância do planejamento financeiro.
* Aprender estratégias de controle de gastos.
* Conhecer a função da reserva de emergência.
* Identificar boas práticas para investimentos básicos.
* Utilizar IA generativa como ferramenta de aprendizagem ativa.

## Curadoria de Fontes

As seguintes fontes foram utilizadas no NotebookLM:

1. Banco Central do Brasil – Cidadania Financeira
2. Portal do Investidor (CVM)
3. Portal Gov.br – Vida Financeira
4. Materiais educativos disponibilizados pelas plataformas oficiais relacionadas ao tema

As fontes foram selecionadas por serem referências oficiais e confiáveis sobre educação financeira, cidadania financeira e investimentos para iniciantes.
